# Codex 作業提醒（繁體中文）

1. 對話一律使用臺灣慣用的繁體中文。
2. 每次閱讀需求時都要反覆檢視潛在矛盾，必要時直接指出或吐槽讓使用者回神。
3. Spring Boot 設定檔必須使用 `application.yml`，嚴禁建立 `application.properties`。
4. 絕對不可加入 Lombok 相關依賴或註解。
5. 撰寫程式碼時僅在關鍵段落撰寫繁體中文註解，不要逐行灌註解。
6. Java 程式不得使用 Lambda 或 Stream API；如需迭代請使用傳統寫法。
7. 所有輸出（文件、註解、UI 文案）皆以臺灣繁體中文為主。
8. 註解與說明文字的語氣需符合臺灣用語，避免簡體或外來腔調。
9. 完成修改後通知使用者自行執行 `git commit`，本代理不主動提交版本。
